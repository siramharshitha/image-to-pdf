The Image to PDF Converter is a simple Python-based application that converts multiple image files into a single PDF document. This project uses the Pillow (PIL) library to read image files, process them, and merge them into one PDF file.

The main purpose of this project is to make it easy to combine several images, such as scanned documents, notes, receipts, certificates, or photographs, into a single PDF for easy sharing, storage, and printing.

🔹 How It Works
The user places image files (PNG, JPG, JPEG) in the project folder.
The Python script reads each image using the Pillow library.
Images are converted into RGB format to ensure PDF compatibility.
All selected images are merged in sequence.
A single PDF file named output.pdf is generated.
🔹 Key Features
Converts multiple images into one PDF.
Supports common image formats like PNG, JPG, and JPEG.
Easy to use and lightweight.
Fast PDF generation process.
Suitable for beginners learning Python file handling.
🔹 Technologies Used
Python 3
Pillow (PIL) Library
🔹 Applications
Creating PDF reports from images.
Combining scanned documents.
Storing certificates and notes in PDF format.
Sharing multiple images as a single file.

This project demonstrates the practical use of Python for file handling and image processing. It provides an efficient way to convert and organize multiple images into a professional PDF document with minimal effort. It is an excellent beginner-friendly project for understanding Python libraries and document automation.
